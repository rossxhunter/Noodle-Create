# Noodle Create
Noodle is a collaborative, educational programming language.

Data Types
---
int, float, string, char, bool

Variable Declaration
---

[Data Type] [Variable Name] = [Value]

    int i = 8
    
Numerical Operations
---

- \+ (addition)
- \* (multiplication)
- / (division)
- \- (subtraction)

Logical Operations
---

- && (and)
- || (or)

String Operations
---

- \+ (concatenation)
